## 🚀 COMMENT UTILISER

### Méthode 1 : Via le Tableau de Bord
1. Aller dans **"Gestion des Notes"**
2. Cliquer sur la carte **"Importer des Notes"** (carte orange en bas)
3. Vous êtes sur la page d'importation !

### Méthode 2 : Via la Page Consulter
1. Aller dans **"Consulter les Notes"**
2. Sélectionner une classe
3. Cliquer sur le bouton vert **"Importer Notes"** dans la barre d'outils
4. Vous êtes sur la page d'importation !

### Méthode 3 : URL Directe
```
https://www.myschoolgn.space/notes/importer/
```
