📊 COMPORTEMENT
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

AVEC PERMISSION :
• Voit la case "Suppression définitive"
• Si cochée → SUPPRESSION TOTALE + Log dans SystemLog
• Si décochée → Soft delete (statut DÉMISSIONNAIRE)

SANS PERMISSION :
• Pas de case à cocher
• Soft delete automatique (statut DÉMISSIONNAIRE)
• Message d'information

🗑️ TRAÇABILITÉ

SUPPRESSION DÉFINITIVE :
• Table : administration_systemlog
• Action : SUPPRESSION_DEFINITIVE_ENSEIGNANT
• Détails : ID, nom, école, salaires, affectations sauvegardés

SOFT DELETE :
• Table : utilisateurs_journalactivite
• Action : DESACTIVATION
• L'enseignant reste dans la base avec statut DÉMISSIONNAIRE
