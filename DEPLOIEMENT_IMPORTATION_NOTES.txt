# 8. Nettoyage des caches

sleep 5

⚠️ PROBLÈMES POSSIBLES:

1. ModuleNotFoundError: No module named 'pandas'
   Solution: pip install pandas openpyxl

2. ImportError: No module named 'openpyxl'
   Solution: pip install openpyxl

3. 404 Not Found /notes/importer/
   Solution: Vérifier git pull, redémarrer uWSGI

4. Erreur AJAX (matières ne se chargent pas)
   Solution: Vérifier console navigateur, logs uWSGI

5. Template ne se télécharge pas
   Solution: Vérifier permissions fichiers temporaires
