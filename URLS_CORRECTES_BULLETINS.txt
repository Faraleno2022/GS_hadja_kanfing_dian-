🚀 PROCHAINES ÉTAPES
═══════════════════════════════════════════════════════════════════════

1. Créer les notes mensuelles:
   python gerer_notes_mensuelles.py --auto

2. Copier-coller une URL ci-dessus dans votre navigateur

3. Vérifier que vous voyez:
   • Le nom de l'élève
   • Le tableau des notes avec 1 colonne (mensuel) ou 2 colonnes (trimestre)
   • La moyenne générale
   • La mention

4. Tester l'impression: Ctrl+P

📞 EN CAS DE PROBLÈME
═══════════════════════════════════════════════════════════════════════

Notes ne s'affichent pas:
  → Vérifier que l'URL contient TOUS les paramètres
  → Vérifier que periode est en MAJUSCULES (OCTOBRE)
  → Vérifier que system_type = mensuel (pas mensuelle)
  → Vérifier que le port est bien 8000

2 colonnes au lieu d'1 (bulletin mensuel):
  → system_type doit être 'mensuel' (pas 'trimestre')

"Période invalide":
  → Utiliser MAJUSCULES: OCTOBRE (pas octobre)
