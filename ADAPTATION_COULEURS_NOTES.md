# Adaptation des Couleurs - Module Notes

## 🎨 Thème appliqué : Bleu/Noir

Le module Notes a été adapté pour correspondre au thème bleu/noir utilisé dans le reste du système.

## 📋 Modifications effectuées

### 1. Tableau de bord Notes (`/notes/`)

#### Cartes de navigation
Les 6 cartes ont été adaptées avec des nuances de bleu et noir :

- **Gérer les Classes** : Bleu principal (#007bff)
- **Gérer les Matières** : Bleu foncé (#0056b3)
- **Gérer les Élèves** : Bleu marine (#1e3a5f)
- **Saisir les Notes** : Bleu Bootstrap (#0d6efd)
- **Générer Bulletins** : Bleu très foncé (#003d82)
- **Statistiques** : Noir/gris foncé (#1a1a2e)

#### En-tête
- Dégradé bleu : `linear-gradient(135deg, #007bff 0%, #0056b3 100%)`
- Ombre bleue : `box-shadow: 0 5px 20px rgba(0, 123, 255, 0.3)`

### 2. Page Gérer les Classes (`/notes/classes/`)

#### Cartes de statistiques
- **Total des classes** : Dégradé bleu `#007bff → #0056b3`
- **Classes actives** : Dégradé bleu foncé `#0d6efd → #003d82`
- Ombre bleue ajoutée pour cohérence

#### Badges de statut
- **Badge Actif** : Dégradé bleu `#007bff → #0056b3` avec ombre bleue
- **Badge Inactif** : Dégradé gris `#6c757d → #495057` avec ombre grise

#### En-tête du tableau
- Dégradé bleu : `linear-gradient(135deg, #007bff 0%, #0056b3 100%)`

#### Boutons
- **Bouton Principal** : Dégradé bleu avec effet hover et élévation
- **Bouton Modifier** : Dégradé bleu clair
- **Bouton Supprimer** : Dégradé gris (au lieu de rouge)

## ✨ Effets visuels

### Ombres
- Ombres bleues pour les éléments bleus : `rgba(0, 123, 255, 0.2-0.4)`
- Ombres grises pour les éléments neutres : `rgba(108, 117, 125, 0.3)`

### Transitions
- Effet d'élévation au survol des boutons
- Dégradés de couleur au survol
- Transitions fluides de 0.3s

### Dégradés
- Direction : 135deg (diagonal)
- Progression naturelle du clair au foncé
- Cohérence avec le reste du système

## 🔄 Cohérence avec le système

Les couleurs sont maintenant alignées avec :
- Module Salaires (bleu #007bff → #0056b3)
- Module Paiements (bleu principal)
- Module Rapports (bleu)
- En-têtes globaux (dégradés bleus)
