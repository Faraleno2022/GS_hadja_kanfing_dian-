========================================
🔧 DÉPANNAGE
========================================

PROBLÈME: Les images ne se chargent pas
SOLUTION:
1. Vérifier que collectstatic a réussi
2. Vérifier les permissions des fichiers
3. Redémarrer l'application
4. Vider le cache du navigateur (Ctrl+Shift+R)

PROBLÈME: Les images sont encore lourdes
SOLUTION:
1. Vérifier dans DevTools que les .webp sont chargés
2. Si .jpg sont chargés, vérifier le navigateur
3. Vider le cache et recharger

PROBLÈME: Erreur 404 sur les images
SOLUTION:
1. Vérifier que les fichiers sont dans static/images/optimized/
2. Réexécuter: python manage.py collectstatic --noinput
3. Vérifier STATIC_URL dans settings.py
