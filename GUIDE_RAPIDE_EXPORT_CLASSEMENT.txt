📊 FONCTIONNALITÉ
─────────────────────────────────────────────────────────────────────────────
Exporter les classements des élèves avec:
  • Rang (avec médailles 🥇🥈🥉)
  • Matricule
  • Nom Complet
  • Moyenne /20

🚀 UTILISATION EN 5 ÉTAPES
─────────────────────────────────────────────────────────────────────────────
1️⃣  Accéder à: http://127.0.0.1:8000/notes/consulter/

2️⃣  Sélectionner une CLASSE dans le formulaire

3️⃣  (Optionnel) Appliquer des FILTRES:
    • Matière (pour classement par matière)
    • Période (mois, trimestre, semestre)
    • Type (mensuelle, composition, etc.)

4️⃣  Cliquer sur le bouton "EXPORTER CLASSEMENT" 🏆 (jaune)

5️⃣  Choisir le type d'export:
    • Classement Général (toutes matières)
    • Par Matière (matière filtrée)

✅ Le fichier Excel se télécharge automatiquement!

🎨 MISE EN FORME AUTOMATIQUE
─────────────────────────────────────────────────────────────────────────────
Podium:
  🥇 1ère place → Fond jaune
  🥈 2ème place → Fond bleu clair
  🥉 3ème place → Fond orange clair

Moyennes:
  ≥ 16 → Vert (Très Bien)
  ≥ 14 → Bleu (Bien)
  ≥ 10 → Jaune (Passable)
  < 10 → Rouge (Insuffisant)

💡 EXEMPLES D'UTILISATION
─────────────────────────────────────────────────────────────────────────────
Exemple 1: Classement mensuel d'Octobre
  Classe: 1ère année
  Période: Octobre
  Export: Classement Général
  → Fichier: Classement_1ère_année_20241103_143025.xlsx

Exemple 2: Classement en Mathématiques
  Classe: 7ème Année
  Matière: MATHÉMATIQUES
  Période: 1er Trimestre
  Export: Par Matière
  → Fichier: Classement_7ème_Année_20241103_143045.xlsx

Exemple 3: Composition du 1er Trimestre
  Classe: Terminale
  Type: Composition
  Période: 1er Trimestre
  Export: Classement Général
  → Fichier: Classement_Terminale_20241103_143105.xlsx

🔧 EN CAS DE PROBLÈME
─────────────────────────────────────────────────────────────────────────────
❌ Bouton non visible
   → Vérifier qu'une classe est sélectionnée

❌ Export vide ou erreur
   → Vérifier que des notes sont saisies pour la période

❌ Fichier ne se télécharge pas
   → Vérifier votre navigateur (autoriser les téléchargements)

❌ Erreur "openpyxl"
   → Installer: pip install openpyxl
