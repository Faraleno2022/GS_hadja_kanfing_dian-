MÉTHODE DE CALCUL DES NOTES - SYSTÈME {{ system_type|upper }}

{% if system_type == 'mensuel' %}

📊 Calcul Mensuel : La moyenne est calculée à partir de toutes les évaluations continues du mois.

📈 Formule : Moyenne = Somme des notes / Nombre d'évaluations

{% elif system_type == 'trimestre' %}

📊 Calcul Trimestriel : La moyenne combine les évaluations continues et la composition de fin de trimestre.

📈 Formule : Moyenne = (Moy. Continue + Composition) / 2

⚖️ Pondération : La composition et les contrôles continus ont le même poids (50% chacun)

{% elif system_type == 'semestre' %}

📊 Calcul Semestriel : La moyenne combine les évaluations continues et l'examen semestriel.

📈 Formule : Moyenne = (Moy. Continue + Examen) / 2

⚖️ Pondération : L'examen semestriel et les contrôles continus ont le même poids (50% chacun)

{% elif system_type == 'annuel' %}

📊 Calcul Annuel : La moyenne annuelle est calculée à partir des moyennes des deux semestres.

📈 Formule : Moyenne = (Moy. 1er Semestre + Moy. 2ème Semestre) / 2

{% endif %}

🎯 Moyenne Générale : (Σ (Moyenne × Coefficient matière)) / Σ Coefficients

📋 Mentions : Excellent ≥18.5 | Très bien ≥16.5 | Bien ≥14.5 | Assez bien ≥12.5 | Passable ≥10 | Faible ≥9 | Insuffisant <9
