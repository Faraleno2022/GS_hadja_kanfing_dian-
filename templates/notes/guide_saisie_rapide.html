Comment Utiliser (3 Étapes Simples)

1

Accéder à l'Interface

Menu → Gestion de notes → Choisir classe → Choisir matière → Créer/Sélectionner évaluation → "Saisir les notes (Simple)"

URL directe : /notes/evaluations/{evaluation_id}/saisie-simple/
2

Saisir les Notes

Deux méthodes au choix :

Méthode 1 : Saisie Manuelle
  1. Cliquez dans le champ "Note /20"
  2. Tapez la note (ex: 15.5)
  3. Appuyez sur Entrée
  4. Passez automatiquement au suivant
Méthode 2 : Pré-remplissage
  1. Cliquez "Remplir tout à 10"
  2. Toutes les notes = 10/20
  3. Modifiez les notes différentes
  4. Gain de temps énorme !
3

Enregistrer

Cliquez sur le bouton flottant "Enregistrer les Notes" (en bas à droite). Le badge indique le nombre de notes à sauvegarder. Confirmation automatique !
