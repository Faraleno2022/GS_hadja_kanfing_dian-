Comment ça marche ?
  1. Téléchargez le template : Sélectionnez une classe et une période, puis cliquez sur "Télécharger Template"
  2. Remplissez les notes : Le fichier Excel contient tous les élèves et toutes les matières de la classe
  3. Importez le fichier : Uploadez le fichier rempli et le système récupère automatiquement toutes les notes
Étape 1: Télécharger le Template
Format du Template
Le fichier contient :
• N° - Numéro d'ordre
• Matricule - Identifiant unique
• Prénoms - Prénoms de l'élève
• Nom - Nom de famille
• Sexe - M ou F
• Matières - Une colonne par matière
