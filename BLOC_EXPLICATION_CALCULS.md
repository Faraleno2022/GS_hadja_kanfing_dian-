## 📊 Contenu du Bloc

### Titre
```
🧮 Méthode de Calcul des Notes
```

### Sections Incluses

**1. Période Évaluée**
```
📋 Période évaluée : 1er Trimestre (Octobre, Novembre, Décembre + Composition)
```
→ S'adapte automatiquement selon la période sélectionnée

**2. Calcul de la Moyenne par Matière**
```
🔢 Calcul de la moyenne par matière :
1. Moyenne mensuelle = (Note mois 1 + Note mois 2 + Note mois 3) ÷ 3
2. Moyenne de la matière = (Moyenne mensuelle + Note composition) ÷ 2
3. Points de la matière = Moyenne de la matière × Coefficient
```

**3. Calcul de la Moyenne Générale**
```
📊 Calcul de la moyenne générale :
• Total des points = Somme des points de toutes les matières
• Total des coefficients = Somme des coefficients de toutes les matières
• Moyenne générale = Total des points ÷ Total des coefficients
```

**4. Attribution du Rang**
```
🏆 Attribution du rang :
• Les élèves sont classés par ordre décroissant de moyenne générale
• Le rang est attribué en fonction de ce classement
• En cas d'égalité (ex-aequo), les élèves partagent le même rang
```

**5. Attribution de la Mention**
```
🎖️ Attribution de la mention :
[Très Bien]    Moyenne ≥ 16/20
[Bien]         Moyenne ≥ 14/20 et < 16/20
[Assez Bien]   Moyenne ≥ 12/20 et < 14/20
[Passable]     Moyenne ≥ 10/20 et < 12/20
[Insuffisant]  Moyenne < 10/20
```
→ Avec badges colorés

**6. Exemple Concret**
```
📝 Exemple de calcul pour une matière :

Français (Coefficient 4) :
• Notes mensuelles : Octobre = 14, Novembre = 15, Décembre = 13
• Moyenne mensuelle = (14 + 15 + 13) ÷ 3 = 14
• Note composition = 16
• Moyenne matière = (14 + 16) ÷ 2 = 15/20
• Points = 15 × 4 = 60 points
```

**7. Note Officielle**
```
ℹ️ Ce système de notation suit le programme officiel du système éducatif guinéen.
```

## 🎓 Exemple Complet

### Données
```
Français (Coef 4):
- Octobre: 14
- Novembre: 15
- Décembre: 13
- Composition: 16

Mathématiques (Coef 4):
- Octobre: 12
- Novembre: 13
- Décembre: 14
- Composition: 15
```

### Calculs

**Français**:
```
1. Moyenne mensuelle = (14 + 15 + 13) ÷ 3 = 14
2. Moyenne matière = (14 + 16) ÷ 2 = 15
3. Points = 15 × 4 = 60
```

**Mathématiques**:
```
1. Moyenne mensuelle = (12 + 13 + 14) ÷ 3 = 13
2. Moyenne matière = (13 + 15) ÷ 2 = 14
3. Points = 14 × 4 = 56
```

**Moyenne Générale**:
```
Total points = 60 + 56 = 116
Total coefficients = 4 + 4 = 8
Moyenne générale = 116 ÷ 8 = 14.5/20
Mention: Bien
```
