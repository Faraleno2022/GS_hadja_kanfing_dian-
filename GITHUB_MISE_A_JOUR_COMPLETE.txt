UTILISATION :
═════════════

Interface web :
  1. Aller sur /notes/consulter/
  2. Sélectionner classe et période
  3. Cliquer "Exporter Classement" 🏆
  4. Choisir "Format PDF" → "Classement Général" ou "Par Matière"

URLs directes :
  • Général PDF : /notes/exporter-classement-pdf/?classe_id=X&periode=Y
  • Matière PDF : /notes/exporter-classement-pdf/?classe_id=X&matiere_id=Z&periode=Y
  • Général Excel : /notes/exporter-classement/?classe_id=X&periode=Y
